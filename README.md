# Tank-Example
這是 2025 Rotational Training 第六/七次培訓課程的示範程式。

## 關於 Robot[_Complete].java
在 `src/main/java/frc/robot/` 內有 `Robot.java` 及 `Robot_Complete.java` 等檔案。\
`Robot_Complete.java` 有包含各種搖桿、廠商的示範程式碼，並以註解取消其作用，做為不同情況下的參考。\
`Robot.java` 則去除了多於註解，僅保留了使用「Xbox Controller」為搖桿、「CTRE Victor SPX」為馬達控制器的程式碼。\
⚠️ **實際執行程式時，機器人會執行 `Robot.java` 的內容而非 `Robot_Complete.java`。**
